# Mantle Sentinel
## Tagline
AI agents that observe, predict, debate, remember, and evolve.
---
## Project Type
Hackathon MVP
Primary Goal:
Build a living ecosystem of autonomous analyst agents that:
- observe events
- generate predictions
- debate each other
- maintain memory
- evolve over time
The system should feel alive even when no human is interacting with it.
---
## Core Vision
Most AI demos are stateless.
Mantle Sentinel attempts to demonstrate:
- persistence
- memory
- disagreement
- adaptation
- evolution
Every agent has:
- personality
- confidence
- prediction history
- memory graph
- evolution history
Agents are not chatbots.
Agents are independent actors.
---
## MVP Objective
Demonstrate:
1. Multi-agent prediction generation
2. Agent disagreement
3. Memory accumulation
4. Evolution events
5. Alert generation
Everything visible through a lightweight dashboard.
---
## Non Goals
Not included in MVP:
- Dream Mode
- BOCPD
- Long-term autonomous planning
- Production scaling
- Distributed memory
- Reinforcement learning
- Agent self-modification
- Fine tuning
---
## Hackathon Success Criteria
A judge can:
- open dashboard
- see multiple agents
- inspect predictions
- inspect disagreements
- inspect memory growth
- inspect evolution events
- inspect alerts
within 2 minutes.
---
## Target Audience
Hackathon judges
AI engineers
Investors
Researchers
Builders
---
## Technical Philosophy
Prefer:
- simplicity
- observability
- determinism
- reliability
Avoid:
- unnecessary AI complexity
- hidden magic
- research rabbit holes
---
## MVP Principle
Visible intelligence beats theoretical intelligence.
If judges cannot see it,
it does not exist.
