The problem

Traditional monitoring is blind to novel attacks

Rule-based systems and signature scanners only catch what they've seen before. DeFi exploits are designed to evade them.

Exploits look normal at Tx level

Flash loan attacks, reentrancy, and price manipulation use valid function calls. Signature scanners see nothing unusual.

No training data for zero-days

ML models trained on historical exploits have 0% recall on novel attack vectors. The next exploit is always the first of its kind.

Slow drift evades static thresholds

"Boil the frog" attacks gradually shift contract behavior over hundreds of transactions — staying below every fixed alert threshold until the damage is done.

How it works

5-tier detection spine

Each tier filters a different attack surface. Hard alerts fan out at Tier 1; behavioral drift catches everything else.

Tier 0
Spam pre-filter
Timing flood — k=20 consecutive sub-interval txs → spam_attack. Excluded from window.
Tier 1
Entropy filter
Per-selector Shannon baseline. |H − mean| > 4σ → entropy_anomaly. Hard alert, instant.
Tier 2
HDC Encoder
D=10,000-dimensional bipolar hypervector. Role-filler binding over sliding window W=50.
Tier 3
Drift signal
drift = max(Hamming(V,P)/D, |z_timing|). Robust median/MAD normalization.
Tier 4
Detector
Static (θ=0.65, k=3) or BOCPD — Bayesian Online Change Point Detection for slow drift.
Tier 5
Interpreter + Fan-out
Feature-ablation attribution. Alert → Z.ai explain → Telegram → on-chain anchor.

Capabilities

Built for production DeFi security

Hyperdimensional Computing

D=10,000 bipolar vectors encode behavioral patterns mathematically. No training, no labels, no model weights to steal or manipulate.

Explainable by Z.ai

Every alert comes with a human-readable attribution. "Selector distribution changed 61%, gas elevated 22%." Not a black box.

Dream Mode

Prototype self-heals. Every 100 safe windows, the behavioral baseline consolidates (P_new = sign(λ·P_old + ΣV_safe)) — without retraining.

BOCPD for slow drift

Bayesian Online Change Point Detection catches gradual attacks before static thresholds trigger. Plug-in replacement via SENTINEL_DETECTOR=bocpd.

On-chain immutable record

SentinelAlertRegistry logs every alert to Mantle mainnet. Tamper-proof audit trail for post-incident forensics and insurance claims.

Deterministic & auditable

MASTER_SEED=1337. Every benchmark is reproducible. 109 tests pass from a clean checkout. No hidden state, no stochastic inference at runtime.
